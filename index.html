Que es HTML?

(HyperText Markup Language) es el lenguaje estándar utilizado para crear y diseñar páginas web. Es un lenguaje de marcado que se utiliza para estructurar el contenido de una página web, incluyendo texto, imágenes, enlaces, formularios y otros elementos multimedia.

Fundamentos de HTML

  1. Etiquetas y elementos:

    HTML utiliza etiquetas para definir diferentes tipos de contenido en una página web. Por ejemplo, la etiqueta <p> se utiliza para definir un párrafo de texto, <h1> a >h6> se utilizan para encabezados de diferentes tamaños, <img> se utiliza para insertar imágenes, etc. </p>

  2. Estructura básica:

    Toda página HTML comienza con la etiqueta <html> y tiene dos secciones principales: <head> y <body>. El <head> contiene metadatos, como el título de la página y enlaces a archivos externos, mientras que el <body> contiene el contenido visible de la página.

  3. Atributos:

    Las etiquetas HTML pueden tener atributos que proporcionan información adicional sobre el elemento. Por ejemplo, la etiqueta <img> tiene el atributo src que especifica la ruta de la imagen a mostrar.</p>

  4. Enlaces:

    Los enlaces (<a>) se utilizan para crear hipervínculos a otras páginas web o recursos en la misma página.</p>

  5. Listas:

    HTML permite crear listas ordenadas (<ol>), listas desordenadas (<ul>), y listas de definición (<dl>).

  6. Formularios:

    Los formularios (<form>) se utilizan para recopilar datos del usuario. Pueden contener campos de entrada de texto, botones, casillas de verificación, botones de opción y más.

  7. Comentarios:

    Puedes incluir comentarios en tu código HTML utilizando la sintaxis <!-- comentario -->. Los comentarios son útiles para hacer notas sobre tu código para referencia futura o para explicar partes del código a otros desarrolladores.

Encabezados

Los encabezados en HTML son elementos utilizados para definir los títulos o subtítulos de una página web. Estos elementos se utilizan para estructurar y organizar el contenido de una manera jerárquica, lo que ayuda a los usuarios y a los motores de búsqueda a comprender la estructura y la importancia relativa de diferentes secciones de la página.


En HTML, los encabezados se representan mediante las etiquetas <h1> a <h6>, donde <h1> es el encabezado más importante y <h6> es el menos importante. El texto dentro de estas etiquetas se visualiza generalmente con un tamaño de fuente más grande y/o en negrita, dependiendo de cómo se haya definido con CSS.


Por ejemplo:

  1. <h1>: se utiliza para el título principal de la página.
  2. <h2>: se utiliza para subtítulos importantes que están subordinados al título principal.
  3. <h3> a <h6>: se utilizan para títulos y subtítulos adicionales con una jerarquía de importancia descendente.

Parrafos

Los párrafos en HTML son elementos utilizados para agrupar y presentar texto en bloques coherentes. Estos elementos permiten estructurar el contenido textual de una página web, lo que facilita la lectura y comprensión del mismo por parte de los usuarios.


En HTML, los párrafos se representan mediante la etiqueta <p>. El texto contenido dentro de la etiqueta <p> se considera un párrafo y se muestra generalmente con un espaciado vertical adicional tanto antes como después del mismo, para indicar claramente dónde comienza y termina el párrafo.


Aquí tienes un ejemplo de cómo se utiliza la etiqueta de párrafo en HTML:

  1. <p>Este es un parrafo de texto</p>
  2. <p>Este es otro parrrafo de texto</p>

Enlaces

Los enlaces en HTML son elementos que permiten a los usuarios navegar de una página web a otra o acceder a diferentes secciones de la misma página web. También se les conoce como hipervínculos. Los enlaces se crean utilizando la etiqueta <a> (anchor), que tiene un atributo href (hipertext reference) que especifica la dirección URL a la que se debe dirigir el enlace.


Aquí tienes un ejemplo de cómo se utiliza la etiqueta de enlace en HTML:

  1. Estructura de la etiqueta de enlace:

  2. <a href="https://google.com">Visita Google Aqui</a>
  3. Visita Google Aqui

Listas

En HTML, las listas son elementos utilizados para organizar y presentar información de manera estructurada y jerárquica. Hay tres tipos principales de listas en HTML:

  1. Listas Ordenadas (<ol>):

    Representan una secuencia de elementos ordenados y se muestran con números o letras. Cada elemento de la lista se define mediante la etiqueta <li> (elemento de lista). El navegador asigna automáticamente los números o letras según el orden en que se definen los elementos <li>.

  2. Ejemplo:

                        
<ol>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
    <li>Elemento 3</li>
</ol>

  3. Listas Desordenadas (<ul>):

    Representan una colección de elementos sin orden específico y se muestran con viñetas o puntos. Al igual que con las listas ordenadas, cada elemento de la lista se define mediante la etiqueta <li>.

  4. Ejemplo:

    
<ul>
        <li>Elemento x</ul>
        <li>Elemento x</ul>
        <li>Elemento x</ul>
</ul>

  5. Listas de Definición (<dl>):

    Consisten en términos (definiciones) y sus respectivas descripciones. Cada término se define mediante la etiqueta <dt> (término de definición), y cada descripción se define mediante la etiqueta <dd> (descripción de definición).

  6. Ejemplo:

    <dl>
        <dt>Término 1</dt>
        <dd>Descripción de Término 1</dd>
        <dt>Término 2</dt>
        <dd>Descripción de Término 2</dd>
</dl>

Tablas

En HTML, las tablas son elementos utilizados para organizar y presentar datos en filas y columnas, de manera similar a una hoja de cálculo. Permiten representar información de forma estructurada y ordenada, lo que facilita la lectura y comprensión de los datos por parte de los usuarios.


Las tablas en HTML se crean utilizando una combinación de etiquetas específicas, que incluyen: 

  1. <table>: Define una tabla en HTML.
<tr>: Define una fila en la tabla.
<td>: Define una celda de datos regular en la fila.
<th>: Define una celda de encabezado en la fila.

Formularios

En HTML, los formularios son elementos que permiten a los usuarios enviar datos a un servidor web para su procesamiento. Los formularios son una parte fundamental de muchas páginas web interactivas, ya que permiten a los usuarios ingresar información y realizar acciones, como enviar mensajes, realizar búsquedas, registrarse en un sitio web, completar encuestas, entre otras cosas.
